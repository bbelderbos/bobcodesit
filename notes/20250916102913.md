Did you know you can make your regex patterns more readable with re.VERBOSE (re.X) so you can split them across lines and add comments. ðŸ’¡ðŸ“ˆ

```
import re

text = """
We launched PyBites on 2016-12-19.
We started coaching around 2020-02-01 ðŸ“ˆ
It's almost Hacktoberfest: join us starting 2025-10-01!
No match: v2020-02-01beta
"""

# Match YYYY-MM-DD anywhere in the text
pat = re.compile(
    r"""
    \b              # word boundary to prevent false positives like last line
    \d{4}           # 4-digit year
    -               # literal dash
    \d{2}           # 2-digit month
    -               # literal dash
    \d{2}           # 2-digit day
    \b              # word boundary
    """,
    re.VERBOSE,
)

dates = pat.findall(text)
print(dates)  # ['2016-12-19', '2020-02-01', '2025-10-01']
```
