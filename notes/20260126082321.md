# Convert Markdown to HTML with markdown-it-py

I just discovered a neat package to convert Markdown to HTML: markdown-it-py

I used it to send a weekly markdown report, which now looks so much better :)

`markdown-it-py` made it trivial;

```python
from markdown_it import MarkdownIt

md = MarkdownIt()

markdown_content = """
## Weekly Report

- **Status**: Complete
- **Link**: [Dashboard](https://example.com)
"""

html = md.render(markdown_content)
# Returns proper <h2>, <ul>, <strong>, <a> tags
```

Works great with SendGrid, SES, or any email service that accepts HTML content.
