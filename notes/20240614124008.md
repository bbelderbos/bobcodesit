# Use match/case over if/elisf/else

Python 3.10 introduced `match/case` which is a more powerful + readable alternative to `if/elif/else` statements. ðŸ ðŸ˜

It's similar to the `switch` statement in other programming languages. ðŸ’¡

Quick example using it for a simple CLI app below. ðŸ“ˆ

```python
def show_options(options: dict[str, str]) -> None:
    for key, value in options.items():
        print(f"{key}: {value}")
    print()

def main():
    options = {"1": "Add item", "2": "Delete item", "3": "View items", "4": "Exit"}
    while True:
        show_options(options)
        selected_option = input("Choose an option: ")

        match selected_option:
            case "1":
                print("Add item logic here")
                # TODO: Implement add item logic
            case "2":
                print("Delete item logic here")
                # TODO: Implement delete item logic
            case "3":
                print("View items logic here")
                # TODO: Implement view items logic
            case "4":
                print("Goodbye!")
                break
            case _:
                print("Invalid option, please try again")
                continue

if __name__ == '__main__':
    main()
```
